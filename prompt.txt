buat tabel users:
id integer auto increment
name varchar 255 not null
email varchar 255 unique not null
password varchar 255 not null (password di hash dari bcrypt)
created_at timestamp default current_timestamp

buatkan API untuk regsitrasi user baru 

Endpoint: POST /api/users

Request Body:
{
    "name": "John Doe",
    "email": "[EMAIL_ADDRESS]",
    "password": "password"
}

Response Success:
{
    "data": "OK"
}

Response Error:
{
    "message": "Email already exists"
}

Struktur folder di dalam src
- routes
- services: berisi logic

Struktur file di dalam routes
- users.routes.ts

Struktur file di dalam services
- users.service.ts
